Privacy Policy for Listen application

This application does not collect or store personal user data directly.
The output files may be shared only by an intended user action.

The app uses a standard voice recording module and accesses microphone upon user explicit request.

The app does not access device information for functionality and analytics purposes.
No data is stored on external servers by the developer.
The app data is stored together with the app and is removed when the app is uninstalled.
The user has full control of the recorder data files.
Any export of data is done by user's specific command.

None
